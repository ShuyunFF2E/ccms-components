组件库说明

1.组件库提供的组件均以 cc- 作为前缀(ccms-components),如 cc-grid、cc-tooltip。 2.组件库提供的服务均以 $cc 作为前缀, 如 $ccModal。

添加 Components 文档

以 Simple Component 为例

1.在 docs/_data/nav.yml 中 Components 下增加一行 title

2.在 docs/components/ 中创建 simple-component.md，在此编写你的文档内容。

3.常用的示例都已经写在 docs/components/simple-component.md 中，可以参考 Sample component 的效果。

4.编写演示 demo 请 fork base jsfiddle
