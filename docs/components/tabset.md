## Tabset组件

指令： tabset

参数说明：

| 参数名 | 是否必填 | 说明 | 类型 |
| --- | --- | --- | --- |
| active | 否 | 当前选中第几个选项，0开始 | Number |
| type | 否 | 组件类型[smaller] | String |

指令： tab

| 参数名 | 是否必填 | 说明 | 类型 |
| --- | --- | --- | --- |
| title | 是 | tab标题 | String |
| onSelect | 否 | 选中的回调 | Function |
