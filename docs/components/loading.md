## 使用说明

loading组件

### 用法

{% highlight html %}
<loading
    [type="string"]>
	transclude template
</form>
{% endhighlight %}

### 参数说明

|参数|类型|说明|
|-------|--------|--------|
| type|string|loading类型, 不设置则为默认类型, 设为 layer 则为带背景层类型|
